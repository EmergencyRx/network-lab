# Network Lab

Practice environment for networking, Linux services, and security tooling in a controlled, reproducible setup.

## Objectives

- Provide reusable topologies for routing, VLANs, VPNs, and firewalls
- Explore common services such as DNS, DHCP, HTTP, and VPN
- Offer debugging and hardening scenarios
- Support learning without targeting external systems

## Suggested Structure

- topologies/
- scenarios/
- configs/
- scripts/
